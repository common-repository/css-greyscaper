Greyscaper admin CSS for WordPress 2.5+.

== Description ==

Greyscaper admin CSS for WordPress 2.5+. Based on '[Greyscale Paper](http://kuler.adobe.com)' color palette by victorfacius. Features three different color schemes:

* (01) default
* (02) default minor variation
* (03) Escape scheme

== Installation ==

1. Upload `css-greyscaper` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Edit your personal profile and choose your favourite admin color scheme.
